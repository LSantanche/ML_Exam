Introduction

The goal of this project is to use the statistical learning methods introduced in the course to analyse a dataset on the quality of white vinho verde wine samples from the north of Portugal (https://en.wikipedia.org/wiki/Vinho_Verde).

Dataset

Predictor variables/features (except ID) describing physicochemical characteristics for the wine samples:

Variable name Description
ID IDentifier
fixed.acidity Fixed acidity (g(tartaric acid)/dm\(^3\))
volatile.acidity Volatile acidity (g(acetic acid)/dm\(^3\))
citric.acid Citric acid (g/dm\(^3\))
residual.sugar Residual sugar (g/dm\(^3\))
chlorides Chlorides (g(sodium chloride)/dm\(^3\))
free.sulfur.dioxide Free sulfur dioxide (mg/dm\(^3\))
total.sulfur.dioxide Total sulfur dioxide (mg/dm\(^3\))
density Density (g/cm\(^3\))
pH pH
sulphates Sulphates (g(potassium sulphate)/dm\(^3\))
alcohol Alcohol (vol.%)
Response variable or target (based on sensory data):
Variable name Description
quality Categorical variable for high quality (based on score > 6) or average/low quality (based on score \(\ge\) 6)

The data files provided are the following:

File name Description
wine_quality_train.csv training set
wine_quality_test.csv test set (without quality class)

Evaluation

  • The goal is to classify the wines according to their quality. In particular, the classification of high quality wines is considered of primary importance.

  • The final classification model will be evaluated on the test set using the F1 score.

  • Each student may provide up to 3 submissions.

  • The submitted file(s) should be formatted as follows:

    ID,Pred,Prob
3,AvgOrLow,0.15
6,High,0.80
13,AvgOrLow,0.38
...

    where ID is the ID for each wine in the test set, Pred the predicted quality class, and Prob the probability of high quality wine.

  • Each submitted file should be named using the scheme <Surname>_<Name>_submission<num>.csv, where <Surname> and <Name> are the student’s surname and name, and <num> is the submission number.

  • The submission file(s) must be uploaded in moodle.

  • Submission deadline: November 26th, 24:00.

  • In-class discussion & presentation: November 30th, 09:00 (each student should consider at most 25-30 min. for the presentation).
